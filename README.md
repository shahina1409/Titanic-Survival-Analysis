# Titanic Data Science Project

## Overview
This project performs end-to-end Data Science analysis on the Titanic dataset. The objective is to analyze passenger data, identify survival patterns, perform statistical analysis, build a machine learning model, and create an interactive prediction system.

## Features
- Data Cleaning and Preprocessing
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Encoding
- Statistical Analysis
- Machine Learning Model Training
- Survival Prediction System
- Interactive Streamlit Application
- Interactive Dashboard
- PDF Report Generation

## Key Insights

- Female passengers had a significantly higher survival rate.
- First-class passengers survived more frequently.
- Higher fare passengers had better survival chances.
- Passenger class, gender, and fare strongly influenced survival.
